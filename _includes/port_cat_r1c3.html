Java Projects

Overview

This page highlights a collection of Java projects I have worked on, showcasing my programming skills and problem-solving abilities. Each project demonstrates practical implementations of core Java concepts, object-oriented programming (OOP), and various libraries.

All projects are available on my GitHub Repository. Feel free to explore, contribute, or reach out with feedback.

Highlighted Projects

  • School Employee Management System: This Java project implements a simple school management system with multiple classes representing employees, faculty, and department chairs. The program demonstrates object-oriented principles such as inheritance and method overriding.
  • Data Structures and Algorithms: A series of programs implementing various data structures (e.g., linked lists, trees, graphs) and algorithms (e.g., sorting, searching, pathfinding) with performance analysis.
  • Grading Application: A Java program demonstrating the use of arrays, random number generation, sorting, and reversing with three distinct problems. Each problem addresses a specific array operation and outputs relevant results.
  • Game Development: A Java-based card game program that simulates the functionality of a card game, allowing multiple players to participate, interact, and play against a dealer.
  • Custom Utilities: Various small utility programs, such as a unit converter, text file parser, and command-line calculator.

Skills Demonstrated

  • File handling, I/O operations, and exception handling.
  • Unit testing and debugging techniques.
