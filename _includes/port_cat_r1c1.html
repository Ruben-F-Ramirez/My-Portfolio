CPU Design

Project Overview

This project involves designing and implementing a 16-bit Central Processing Unit (CPU) using VHDL. The CPU includes an ALU with arithmetic, logic, and shifting capabilities, data and instruction memory, three 16-bit registers, a control unit, and a 4-to-1 multiplexer.

Key Features

  • Arithmetic operations: Addition, subtraction, multiplication, and increment.
  • Logical operations: AND, OR, XOR, XNOR, NOR, and NAND.
  • Shifting and rotating capabilities: Shift left, shift right, rotate left, and rotate right.
  • Instruction memory with a 24-bit instruction set.
  • Control unit for enabling execution and reset states.

Design Details

The CPU operates with two 16-bit data inputs, a 24-bit instruction input, and a clock signal. Outputs include a 16-bit data output, a 17-bit data memory output, and a carry-out signal for arithmetic operations.

Instructions are executed in three cycles:

  1. Fetch: Load the instruction into the instruction register.
  2. Decode: Determine the operation to execute.
  3. Execute: Perform the operation based on the decoded instruction.

ALU Operations

Test Bench Results

The test bench demonstrates the CPU's functionality, including load, ALU operations, memory read/write, and program execution cycles. Highlights include:

  • Successful addition and subtraction operations.
  • Proper instruction decoding and execution.
  • Memory read and write validations.
